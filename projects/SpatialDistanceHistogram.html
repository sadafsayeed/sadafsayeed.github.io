Spatial Distance Histogram
CUDA C
About the project
This project implements a CUDA-based Spatial Distance Histogram (SDH) for 3D point clouds, computing how many point pairs fall within user-defined distance buckets (e.g., [0,w], [w,2w], etc.). It leverages GPU parallelism by assigning each thread to process a point and uses atomic operations to safely update histogram counts. The program supports flexible CUDA configurations (blocks and threads) and includes a CPU version to compare performance and validate accuracy against the GPU results.
