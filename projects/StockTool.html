About the project
This project is a comprehensive Windows Forms stock analysis tool in C# that processes OHLC/volume data from Daily/Weekly/Monthly CSV files, featuring interactive candlestick charts with volume histograms, automatic gap elimination for daily data, and multi-stock comparison in normalized separate forms. The application implements advanced pattern detection with adjustable margin settings to identify peaks and valleys, automatically categorizes valid waves displayed via dropdown selections with colored rectangle/diagonal visualizations (green for UP, red for DOWN), and enables dynamic rubber-banding selection initiating from extremes to draw real-time Fibonacci retracement levels with live confirmation tracking, all optimized through OOP design and efficient data handling for responsive technical analysis.
